Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b>. (Não semântica)

Nesta frase temos um termo em destaque usando a tag <strong>. (É uma forma semântica, uma tag que dá sentido, no caso para por em Destaque).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i>. (Não semântico).

Nesta frase temos um termo em ênfase usando a tag <em>. (Em forma semântica, com um sentido de ênfase.)

Texto marcado

Podemos criar um texto marcado, como um marcador, usando a tag <mark>. Alteando a cor, colocando dentro da própria tag <mark>, com um parâmetro chamado style="background-color: blue"(Sendo blue o "valor", nome da cor).Sabendo que alteração de cores é CSS.

Criando outro texto marcado agora sem alterar a cor, mantendo a cor padrão, que é o que deve ser feito no HTML, apenas. Também utilizando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande com a tag <big> e um texto pequeno utilizando a tag <small>. Embora que a tag BIG está obsoleta, pois não há semântica (sentido) no uso dela, caso queira um texto maior, é possível aumentar a fonte pelo CSS, já no caso do SMALL ainda há uma semântica, em um texto, por exemplo as letras miudas de um contrato.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Com a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, com a tag <ins>

Existe também o sublinhado com a tag U. (Não semântico)

Texto sobre escrito

Para inserir coisas do tipo x2+3. Utiliza-se a tag <sup> de superscript

Texto sub escrito

Para inserir situações como composições químicas, exemplo, H2O. Utiliza-se a tag <sub> de subscript
